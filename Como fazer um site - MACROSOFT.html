Introdução

Construir um site pode parecer desafiador, mas com as ferramentas certas, qualquer pessoa pode aprender! Neste guia, vamos te mostrar como começar do zero.

Quando falamos de HTML, estamos falando de uma linguagem de marcação de hipertexto que compõe a maior parte das páginas disponíveis na internet. HTML é a sigla, em inglês, para HyperText Markup Language , ou linguagem para marcação de hipertexto. Já um hipertexto é todo texto inserido em um documento para a web. 

A título de exemplo, esse site próprio que você está lendo é desenvolvido a partir de html.

Tags Úteis

Para se desenolver um site, os tags são blocos de construção fundamentais, as quais são escritas entre colchetes angulares <> e ocorrem em pares: um tag de abertura e um tag de fechamento.

Veja abaixo algumas tags HTML essenciais para a criação de um site:

  • <DOCTYPE html>: Define o início de um documento HTML.
  • <html lang= "pt-br">: Informa que o conteúdo é escrito em português do Brasil.
  • <meta charset = "UTF-8">: Informa a codificação que será usado.
  • <head>: Contém metadados, como título e links para scripts e estilos.
  • <body>: Contém todo o conteúdo visível da página.
  • <h1> a <h6>: Usado para cabeçalhos. <h1> é o mais importante.
  • <p>: Define parágrafos.
  • <a>: Define links para outras páginas ou recursos.
  • <img>: Insere imagens no site.
  • <div>: Define seções de um documento.
  • <ul> e <ol>: Criam listas ordenadas e não ordenadas.
  • <form>: Cria formulários para entrada de dados do usuário.
  • <footer>: Define o rodapé da página.
